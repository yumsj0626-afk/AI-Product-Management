# AI-Product-Management

 본 레포지토리는 Duke University의 AI Product Management Specialization 과정을 수강하며, AI 기술을 실제 비즈니스 가치로 전환하기 위한 의사결정 체계와 산출물을 기록하는 공간입니다.
 
 - 학습 동기: AI 모델의 성능이 곧 제품의 성공으로 직결되지 않는다는 문제의식에서 출발하여, 데이터 품질, 모델 드리프트, 윤리적 리스크 등 AI 특유의 불확실성을 관리하는 전문 PM 역량을 확보하고자 합니다.
